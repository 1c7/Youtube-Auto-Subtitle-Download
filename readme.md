#### 为啥有字幕和自动字幕之分?  
在一般网站上 视频只有两种状态...有字幕/没字幕  
但Youtube有语音识别技术(目前只支持英文)  
所以有的视频会有机器识别的自动字幕(英文的), 准确率在70~95%左右, 具体取决于主持人的发音清不清晰..
